태그

  1. 강조
    < strong > < /strong >
  2. 밑줄
    < u > < /u>
  3. 제목
    < h1 ~ h6 > < /h1 ~ h6>
  4. 줄바꿈
    < br >
  5. 단락
    < p > < /p >
  6. 이미지
    < img src="(이미지파일 이름.파일형식)">
  7. 이미지 크기 수정
    < img src="(이미지파일 이름.파일형식)" width="(숫자 or %)">
  8. 목록
    부모
    - 숫자 X < ul > < /ul >
    - 숫자 O < ol > < /ol >
    자식
    - < li >
  9. 웹페이지 제목
    < title > < /title >
  10. 웹브라우저야 이 파일을 열때는 utf-8 로 열어
    < meta charset="utf-8" >
  11. 몸통(본문)
    < body > < /body >
  12. 머리(본문설명)
    < head > < /head >
  13. 최고위태그
    < html > < /html >
  14. 이 문서는 html 입니다
    < !doctype html >
  15. 링크
    < a href="(링크)" >(링크걸말)< /a >
    새탭으로열기 < a href="(링크)" target="_blank" >(링크걸말)< /a >
    링크미리보기 < a href="(링크)" title="(미리보기)" >(링크걸말)< /a >
  16. 체크박스
    < input type="checkbox" >
  17. 동영상 삽입
    < iframe width="(가로높이)" height="(세로높이)" src="(링크)" >< /iframe >
